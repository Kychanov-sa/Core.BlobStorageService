# Изменения сервиса по версиям

Изменения в обратном хронологическом порядке от последних к более ранним.

# Версия 0.1.0

1. Добавлено REST API с поддержкой функции загрузки и чтения блоба, получения метаинформаци по блобу, удаления и восстановления, создания копии.

# Версия 0.2.0
1. Добавлена поддержка автоочистки корзины по расписанию.
1. Добавлена поддержка автоматического удаления пустных директорий хранилища  по расписанию.
1. Добавлена поддержка удаления BLOB объектов с исёкшим сроком действия по расписанию.
